## Километры в мили
---
Напишите функцию для преобразования километров в мили. (Фактор преобразования - 0.621371)  

Входные данные  
Функции передаётся одно вещественное число - значение в километрах.  

Вернуть значение в милях.

### Пример
---
Ввод
```python
print(km2miles(13.0))
```
Вывод
```
8.077823
```
